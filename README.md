# 🌟 Overview

> **MultiScreen** is a modern Flutter application designed to demonstrate smooth navigation between multiple screens with a clean and responsive user interface.

This project helps developers understand:

✨ Navigation Routing  
✨ UI/UX Design  
✨ Flutter Widgets  
✨ Mobile App Structure  
✨ Responsive Layouts  
✨ Stateful & Stateless Widgets

# 🚀 Getting Started

## 📥 Clone Repository

```bash
git clone https://github.com/Vijay11-08/MultiScreen.git
```

---

## 📂 Move to Project Folder

```bash
cd MultiScreen
```

---

## 📦 Install Dependencies

```bash
flutter pub get
```

---

## ▶️ Run Application

```bash
flutter run
```
